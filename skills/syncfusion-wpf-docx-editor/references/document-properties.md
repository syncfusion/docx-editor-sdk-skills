# Document Properties in WPF DOCX Editor

Document properties support in the Syncfusion WPF DOCX Editor allows retrieving document‑level statistics and state information during editing.

---

## Word count

```csharp
int wordCount = richTextBoxAdv.WordCount;
```
---

## Paragraph count

```csharp
int paragraphCount = richTextBoxAdv.ParagraphCount;
```
---

## Page count

```csharp
int pageCount = richTextBoxAdv.PageCount;
```
---

## Current page number

```csharp
int currentPageNumber = richTextBoxAdv.CurrentPageNumber;
```
